# Angular Dashboard (Auth + Users CRUD)

A modern Angular application built with standalone components, Signals-based state management, and a protected dashboard layout.

This project demonstrates real-world Angular architecture patterns including authentication, route protection, optimistic UI updates, and clean feature-based structure.
